# Events in Vadodara

A beautiful one-page website showcasing upcoming events in Vadodara, Gujarat.

## Features

- Responsive design that works on all devices
- Modern gradient background
- Interactive event cards with hover effects
- Clean and minimal UI

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Responsive Design

## Getting Started

Simply open `index.html` in your web browser to view the website.

## Events Showcased

1. Vadodara Music Festival
2. Contemporary Art Exhibition
3. Vadodara Food Festival
4. Classical Dance Performance
5. Tech Innovation Summit
6. Heritage Walk Tour
